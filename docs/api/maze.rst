Maze
====

:py:class:`quest.maze.Maze` is a helper class which generates mazes. It's included
in the Quest framework, but you could easily use it in other projects. 
:py:class:`quest.maze.Maze` is used in the :py:class:`quest.examples.maze.MazeGame`
demo project.
